Simple if Statement

Executes code only when condition is True

if condition:
    # code block when True
age = 20 if age >= 18: print("You are eligible to vote") print("Please register if you haven't already") print("Program continues here")

if-else Statement

Chooses between two paths

if condition:
    # code when True
else:
    # code when False
temperature = 15 if temperature > 20: print("It's warm outside") print("Wear light clothes") else: print("It's cool outside") print("Consider wearing a jacket") print("Enjoy your day!")

if-elif-else Statement

Handles multiple conditions in sequence

if condition1:
    # code 1
elif condition2:
    # code 2
else:
    # default code
score = 85 if score >= 90: print("Grade: A") elif score >= 80: print("Grade: B") elif score >= 70: print("Grade: C") elif score >= 60: print("Grade: D") else: print("Grade: F")

Nested Conditionals

Conditions within conditions for complex logic

if outer_condition:
    if inner_condition:
        # both True
    else:
        # outer True, inner False
else:
    # outer False
is_weekend = True weather = "sunny" if is_weekend: if weather == "sunny": print("Go to the beach") print("Don't forget sunscreen!") else: print("Watch a movie at home") print("Make some popcorn") else: if weather == "sunny": print("Go to work but enjoy the weather during breaks") else: print("Go to work and stay cozy inside")

Comparison & Logical Operators

Comparison

  • == (equal)
  • != (not equal)
  • > (greater than)
  • < (less than)
  • >= (greater or equal)
  • <= (less or equal)

Logical

  • and (both True)
  • or (either True)
  • not (inverts)

Dictionary Mapping

Alternative to switch-case using dictionaries

operations = {
    'add': add_func,
    'subtract': subtract_func
}
result = operations[choice](x, y)
def calculate_add(x, y): return x + y def calculate_subtract(x, y): return x - y def calculate_multiply(x, y): return x * y def calculate_unknown(): return "Unknown operation" operations = { 'add': calculate_add, 'subtract': calculate_subtract, 'multiply': calculate_multiply } operation = 'add' x, y = 10, 5 result = operations.get(operation, calculate_unknown)(x, y) print(f"Result: {result}")
